Privacy Policy

Last updated: May 4, 2026

Portfolio Tracker ("the Extension") is a Chrome browser extension that aggregates investment portfolio data from supported brokerage sites into a single side panel. Your privacy is important to us. This policy explains what data the Extension collects and how it is used.

1. Data We Collect

Portfolio Tracker is designed with a privacy-first approach. All data stays on your device:

  • Portfolio data — Current value, holdings, P&L, and invested value read from supported brokerage sites using your existing login session. This data is stored locally on your device only.
  • Exchange rate — A single API call is made to fetch the current USD/INR exchange rate for currency conversion display. No personal data is sent in this request.
  • Settings — Your display preferences (currency, refresh interval, alert thresholds) are stored locally in Chrome's storage API.

2. Data Stored on Your Device

The following data is stored locally on your device only using Chrome's local storage API:

  • Portfolio snapshots and history (for trend charts)
  • Holdings data per brokerage source
  • User settings and preferences
  • Cached exchange rate

This data never leaves your device and is not transmitted to us or any third party.

3. Permissions Used

  • cookies — Used solely to authenticate API requests to brokerage sites using your existing logged-in session. Cookies are never stored, copied, or transmitted outside the brokerage domain.
  • storage / unlimitedStorage — Used to save portfolio data and history locally on your device.
  • scripting — Used to inject content scripts into supported brokerage sites to read your portfolio data from the page.
  • tabs — Used to detect when you are on a supported brokerage site to trigger a data refresh.
  • alarms — Used to schedule automatic portfolio refresh at your chosen interval (daily, weekly, or monthly).
  • notifications — Used to alert you when your portfolio value changes beyond a threshold you configure. Notifications are generated locally.
  • sidePanel — The Extension's primary UI is displayed in Chrome's side panel.

4. Third-Party Services

  • Exchange rate API — A public exchange rate endpoint is queried periodically to convert portfolio values between INR and USD. No personal data is included in this request.
  • Supported brokerages — The Extension reads data directly from Zerodha, Groww, Paytm Money, MFCentral, Fidelity, INDMoney, NPS, and EPF using your existing authenticated session. No credentials are collected or stored by the Extension.

5. What We Do Not Do

Portfolio Tracker does not:

  • Collect your name, email, phone number, or any personal identifiers
  • Transmit your portfolio data to any external server
  • Store or transmit your brokerage login credentials or cookies externally
  • Use analytics, tracking, or advertising SDKs
  • Execute remote code — all scripts are bundled within the extension package
  • Share or sell any data to third parties

6. Data Deletion

Since all data is stored locally on your device, you can delete it at any time by:

  • Using the "Clear all data" option in the Extension's Settings tab
  • Removing the Extension from Chrome
  • Clearing Chrome's extension storage via chrome://settings/siteData

7. Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated date.
